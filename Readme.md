## VPS/Locally deploy!
```console
Eldian-Network@Debian~ $ apt-get -y update
Eldian-Network@Debian~ $ sudo apt-get install -y libgl1
Eldian-Network@Debian~ $ apt-get -y install git gcc python3-pip python3-venv -y
Eldian-Network@Debian~ $ git clone https://github.com/john-wick00/Yumekoo
Eldian-Network@Debian~ $ cd Yumekoo
Eldian-Network@Debian~ $ python3 -m venv myenv
Eldian-Network@Debian~ $ source myenv/bin/activate
Eldian-Network@Debian~ $ pip3 install -U -r requirements.txt
Eldian-Network@Debian~ $ bash start
```
